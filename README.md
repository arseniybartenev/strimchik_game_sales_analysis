## Описание проекта

Интернет‑магазин «Стримчик» занимается продажей компьютерных игр по всему миру. Из открытых источников доступны исторические данные о продажах игр, оценках пользователей и экспертов, жанрах и платформах.  
Цель проекта — выявить закономерности, определяющие успешность игры, для планирования эффективных рекламных кампаний.

## Данные

Датасет содержит информацию о более чем 16 тысячах игр, выпущенных с 1980 по 2016 год. Каждая запись включает:

- название игры  
- платформа  
- год выпуска  
- жанр  
- продажи в Северной Америке, Европе, Японии и других регионах  
- оценки критиков и пользователей  
- рейтинг ESRB  

Исходный файл: `data/games.csv`.

## Этапы работы

1. **Подготовка и предобработка данных**  
   - Загрузка данных, изучение общей информации, исправление типов, обработка пропусков и дубликатов, добавление столбца с суммарными продажами.

2. **Исследовательский анализ данных (EDA)**  
   - Анализ динамики выпуска игр, популярности платформ, жизненного цикла платформ, влияния оценок на продажи.  
   - Изучение распределения продаж по жанрам.

3. **Портрет пользователя по регионам**  
   - Сравнение предпочтений платформ, жанров и рейтингов в Северной Америке, Европе и Японии.

4. **Проверка статистических гипотез**  
   - Гипотеза о равенстве средних пользовательских рейтингов платформ Xbox One и PC (не отвергнута).  
   - Гипотеза о различии средних рейтингов жанров Action и Sports (отвергнута, различия значимы).

5. **Формулировка выводов и рекомендаций**  
   - На основе анализа предложены рекомендации для рекламных кампаний с учётом региональных особенностей.

## Используемые технологии

- **Python** — основной язык.  
- **Pandas, NumPy** — обработка и анализ данных.  
- **Matplotlib** — визуализация.  
- **SciPy** — статистические тесты.  
- **Jupyter Notebook** — среда разработки.

## Результаты

- Выявлен пик выпуска игр в 2008–2009 гг., после чего наблюдается спад, связанный с изменением подходов к разработке.  
- Средний срок жизни платформы — около 10 лет; пик популярности приходится на первые 5 лет.  
- Оценки критиков оказывают большее влияние на продажи, чем пользовательские.  
- В Северной Америке и Европе лидируют платформы PS4 и XOne, жанры Action, Shooter, Sports, рейтинг M. В Японии — 3DS, жанр Role‑Playing, рейтинги E и T.  
- Модель не строилась, но анализ позволил сформулировать маркетинговые рекомендации.

## Выводы

Для успешного планирования рекламных кампаний следует:
- учитывать региональные предпочтения (Action/Sports для Запада, Role‑Playing для Японии);
- ориентироваться на оценки критиков;
- делать ставку на перспективные платформы (PS4, 3DS);
- адаптировать рекламные сообщения под возрастные рейтинги.

## Запуск проекта

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/arseniybartenev/strimchik_game_sales_analysis.git
   ```
2. Перейдите в папку проекта:
   ```bash
   cd strimchik_game_sales_analysis
   ```
3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
4. Поместите файл `games.csv` в папку `data/`.
5. Запустите Jupyter Notebook:
   ```bash
   jupyter notebook notebooks/analysis_of_computer_game_sales.ipynb
   ```
